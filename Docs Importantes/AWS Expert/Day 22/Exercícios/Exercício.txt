Olá, no exercicio de hoje, vamos criar uma lambda function para testar nossas habilidades.

Definição:
Python, usando boto3
DynamoDB

Essa lambda function, deve ser capaz de gravar e ler informações no dynamodb.
No dynamodb, crie uma tabela para armazenar informações de companhia area:

Table: flights
Items: id, aircraft_prefix, pilot_name, max_load, route
Exemplo do documento:
{id: AA962, aircraft_prefix: N930NN, pilot_name: 'Deep, Jhon', max_load: 136.9, route: GRU-DFW}

Crie uma lambda function para criar itens nessa tabela de flights. 
Crie uma lambda function para listar todos as informações de voos.
Crie uma lambda function para listar informações de um flight id especifico. Use test events como forma de passar a informação do flight id, em json
